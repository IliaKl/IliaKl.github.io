Модель движения волокна (ленты) в вытяжном приборе (между 2 валиками ленточной машины)

2D на canvas - модель сжатия твердоро тела цилиндрами

3D на three js - генерация потока волокон в заданных пределах, по оси z(?) все значения точек по z пересчитывать в соответствии с2D моделью

в верху страницы - форма для ввода значений, кнопки Create, Stop, Run (?)



(по 2D модели) создается 1-мерный массив (дискретизирующий модель: номер ячейки - X, значение в ячейке - Y верхней точки модели ленты (модель делать симметричную - от 0 по Y , и вверх, потом отразить))
при расчете 3D берется X точки, выбирается [~~X] элемент массива, Y точки умнажается на выбранное число.
